This repository was set up to include the google colab scripts developed as part of the lead author's PhD research and currently for a publication in review. The following scripts include:

gradient_boost_magnitude_prediction.ipynb - google colab workbook for predicting canopy height magnitude using an extreme gradient boosting model.
random_forest_magnitude_prediction.ipynb - google colab workbook for predicting canopy height magnitude using a random forest model.
neural_network_magnitude_prediction.ipynb - google colab workbook for predicting canopy height magnitude using an artificial neural network model.
unet_forest_mask.ipynb - google colab workbook for producing the forest area mask using a U-net + ResNet50 encoder backbone model.

Each script is formatted as notebooks, and more specific details on script functionality can be found in the respective files.
